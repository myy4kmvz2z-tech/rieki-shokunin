## 0. 事前準備

- [ ] アプリが正常に起動し、ホームに「利益職人」「販売前β」が表示される
- [ ] 画面レイアウトが端末幅で崩れない（ボタン・入力欄がはみ出さない）
- [ ] ホームから全メニュー（見積作成 / 保存済み見積 / 見積書を印刷 / 請求書を印刷 / 元請管理 / 会社設定）へ遷移できる
- [ ] 各画面の「← 戻る」でホームまたは前画面に戻れる

---

## 1. 見積作成

- [ ] 「＋ 見積を作成」から見積作成画面が開く
- [ ] 元請が未登録の場合、「元請が未登録です…」と表示され入力できない（該当時のみ）
- [ ] **現場:** 現場名を入力できる
- [ ] **現場:** 元請をプルダウンから選択できる
- [ ] **工事:** 工事項目（クロス SP / AA / CF / フロアタイル / シート）を選択できる
- [ ] **工事:** 施工面積（㎡）を数値入力できる
- [ ] **金額:** ㎡単価・材料費・外注費・値引きを入力できる
- [ ] 元請・工事項目を変更すると、材料費が元請単価から自動反映される
- [ ] 上部オレンジカードに **見積利益・利益率** がリアルタイムで更新される
- [ ] 下部に **売上・原価** が表示され、入力変更と連動する
- [ ] 「保存する」を押すと見積一覧へ遷移する
- [ ] 保存後、一覧に新しい見積が表示される
- [ ] 現場名未入力で保存した場合、「名称未設定」として保存される

**備考:**

## 2. 見積編集

- [ ] 見積一覧から「編集」を押すと見積編集画面が開く
- [ ] タイトルが「見積編集」になっている
- [ ] 保存済みの値（現場名・元請・工事・面積・単価・材料費・外注費・値引き・住所・距離）が正しく読み込まれる
- [ ] 値を変更すると利益・売上・原価が再計算される
- [ ] 「上書き保存」を押すと一覧に戻り、変更が反映される
- [ ] 作成日時（`createdAt`）が編集前と同じままである
- [ ] 編集画面から「見積書を印刷」が動作する（印刷セクション参照）
- [ ] 「← 戻る」で一覧に戻り、未保存の変更は反映されない

**備考:**

## 3. 見積削除

- [ ] 見積一覧から「削除」を押すと該当見積が一覧から消える
- [ ] 削除後、ホームの「保存済み見積」件数が減る
- [ ] 削除した見積が「見積書を印刷」「請求書を印刷」の一覧にも出てこない
- [ ] 削除後にブラウザを閉じて再度開いても、削除状態が維持される

**備考:**

## 4. 元請管理

- [ ] 「👥 元請管理」から元請管理画面が開く
- [ ] 初期データ（BRANCHONE / ブルリノベ等）が表示される（初回のみ）
- [ ] **新規追加:** 元請名 + SP/AA/CF/フロア/シート単価を入力して「追加する」で登録できる
- [ ] 元請名が空の場合、アラート「元請名を入力してください。」が出る
- [ ] 同名の元請を追加しようとすると、重複アラートが出る
- [ ] **編集:** 「編集」→ 単価変更 → 「保存」で反映される
- [ ] **編集:** 「キャンセル」で変更が破棄される
- [ ] **削除:** 「削除」→ 確認ダイアログ → OK で削除される
- [ ] 削除後、見積作成の元請プルダウンから消える
- [ ] 元請変更後、見積作成で新元請の材料費が自動反映される

## 5. 会社設定

- [ ] 会社名が空の状態で保存すると、アラート「会社名を入力してください。」が出る
- [ ] 正常保存時、「会社情報を保存しました。」アラートが出る
- [ ] 保存後、見積書・請求書の印刷プレビューに会社情報が反映される
- [ ] インボイス登録番号が請求書に「登録番号：」として表示される

## 10. データ保存（localStorage）

- [ ] 見積を保存 → ブラウザを完全終了 → 再起動 → 見積が残っている
- [ ] 元請を追加・編集 → 再起動 → 変更が残っている
- [ ] 会社設定を保存 → 再起動 → 設定が残っている
- [ ] プライベートブラウジング / シークレットモードでは、終了後にデータが消えることを確認（仕様理解）
